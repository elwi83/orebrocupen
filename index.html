Hur gör jag? (steg-för-steg)
  1. Gå till console.firebase.google.com
  2. Klicka "Lägg till projekt" → ge det ett namn → Skapa projekt
  3. I vänstermenyn: Build → Realtime Database → Skapa databas
  4. Välj plats (europe-west1) → Starta i testläge → Aktivera
  5. Gå till Projektinställningar (kugghjulet)
  6. Scrolla ner till "Dina appar" → klicka på webbikonen </>
  7. Ge appen ett namn → Registrera app
  8. Kopiera firebaseConfig-objektet och klistra in ovan
