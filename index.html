Rum:

Alla som är anslutna till samma rum ser ändringar i realtid.

Hur gör jag? (steg-för-steg)
  1. Gå till console.firebase.google.com och logga in med ett Google-konto
  2. Klicka “Lägg till projekt” → ge det ett namn (t.ex. “orebrocupen”) → Fortsätt → Skapa projekt
  3. I vänstermenyn: Build → Realtime Database → Skapa databas
  4. Välj plats (europe-west1) → Starta i testläge → Aktivera
  5. Gå till Projektinställningar (kugghjulet uppe till vänster)
  6. Scrolla ner till “Dina appar” → klicka på webbikonen </>
  7. Ge appen ett namn → Registrera app
  8. Kopiera firebaseConfig-objektet (allt mellan { och }) och klistra in ovan
  9. Alla som ska samarbeta klistrar in samma konfiguration och samma rumsnamn
